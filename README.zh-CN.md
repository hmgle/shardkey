# ShardKey

[English](README.md)

ShardKey 是一个纯前端的“答题解锁秘密”工具。
你设置问题、答案和门限值（至少答对多少题），系统生成可分享的挑战链接或 JSON 文件。解题者只有答对足够多问题，才能恢复秘密内容。

## 快速开始

直接在浏览器中打开 `index.html` 即可使用：

```bash
open index.html
```

也可以起一个本地静态服务：

```bash
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 核心功能

1. 创建挑战：输入秘密、设置问题与答案、配置门限值
2. 分享挑战：复制带挑战数据的链接，或下载 JSON 文件
3. 求解挑战：通过链接或 JSON 导入，回答问题后尝试解锁

## 技术特性

- 纯 HTML/CSS/JavaScript，零外部依赖
- 使用 BigInt 进行大整数运算
- 使用 Web Crypto API 进行 SHA-256 哈希
- 所有计算都在浏览器本地完成，支持 `file://` 直接打开

## 安全说明

- 分享数据不包含明文秘密、正确答案或答案哈希
- 只有足够数量的正确答案组合才能通过门限恢复秘密
- 校验和用于验证恢复结果，防止错误答案误解锁

### 重要限制

- 公开的校验和会成为**离线猜测的验证口令**：如果你的秘密熵很低（例如短 PIN、手机号等），攻击者可以不答题直接暴力猜测并用校验和验证。需要更强保密性时，请使用高熵秘密（或在秘密后追加随机串）。
- 答案只做了快速 SHA-256 归一化哈希；弱答案容易被离线字典攻击。
- Mignotte/CRT 不是 Shamir 门限方案，也不具备“完美保密”；不要把它当作经过严格审计的密码学秘密共享替代品。
